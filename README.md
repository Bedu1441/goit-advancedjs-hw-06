# goit-advancedjs-hw-06

Homework for Topic 10: TypeScript Typing.

## Description

This project contains practical TypeScript tasks focused on basic types and generics.

The homework includes two main parts:

1. Basic Types
2. Generics

All source files are located in the `src` folder.

## Completed topics

### Basic Types

The following TypeScript concepts were practiced:

- `number`
- `string`
- `boolean`
- `null`
- `undefined`
- `any`
- `unknown`
- tuples
- union types
- literal types
- `enum`
- function return types
- custom object types

### Generics

The following TypeScript generic and utility type concepts were practiced:

- generic functions
- generic classes
- `Promise<T>`
- `Pick`
- `Partial`
- `Record`
- `Omit`
- object merging with generics

Installation

Install dependencies:

npm install
Build

Run TypeScript compilation:

npm run build

If the command finishes without errors, the homework is completed successfully.

Watch mode

For automatic TypeScript compilation during development, run:

npm run watch
Notes

Each TypeScript file contains export {} to make it an isolated module and avoid variable name conflicts between files.

The project uses strict TypeScript checking to practice correct and safe typing.

Source files

The source files are located in the src folder:

src/baseTypes
src/generic
